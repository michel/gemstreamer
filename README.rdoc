= gemstreamer

Very simple Gem that pushes information about gem dependencies in your rails app to gemstream. (http://github.com/ludo/rerr2009/tree/master/public/).
This is a proof of concept for the Ruby En Rails Rumble (http://2009.rubyenrails.nl/geek_dag?locale=en).
